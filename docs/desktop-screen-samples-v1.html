What to check (5 minutes)

  1. Web-first layout — header nav + multi-column desktop layouts. Is this the "web application" you meant?
  2. Data presentation — sparklines + change table (screen 1), forecast band chart + table view (screen 2), shared-scale comparison bands (screen 3). Enough? Too much?
  3. Honest uncertainty everywhere — no bare price is ever shown without its range, confidence word, and reason.
  4. Provenance line under every data view (HARTI / Dambulla) — the #1 trust builder from research.
  5. Confidence wording — "Good / Fair / Low" (Sinhala: හොඳයි). Confirm or redline.
