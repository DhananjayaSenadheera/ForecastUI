AgriForecast — Design System v1

FE-1 · design system & trilingual typography · light-only (Release 1) · every PRD component in its key states, each with a trilingual sample, do/don't and a11y notes. Inline CSS mirrors design-tokens-v1.css; FE-2 imports the token file and rebuilds these as React primitives.

Sinhala & Tamil strings on this page are drafts pending native-speaker review (PRD risk + API gap #5: reason codes come from the API, client owns translation). Emoji stand in for the ~30 commissioned semi-abstract crop illustrations. Rendered in your OS fonts — the app self-hosts the current Noto Sans Sinhala/Tamil builds at FE-2 (never an old CDN binary: the pre-2020 Sinhala wrong-ascender defect clips ascenders).
